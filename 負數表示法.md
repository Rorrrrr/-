```
signed-mangnitude
  step1:判斷正負
        判斷最左側的第一個數字是1or0，1為負數，0為正數
  step2:計算大小(magnitude)
        用2進位數字換算成10進位數字
  step3:完成後正負號+大小數字 
```
```
1's complement
   step1:判斷正負
         判斷最左側的第一個數字是1or0，1為負數，0為正數
   step2:計算大小(magnitude)
    step2-1:先執行反向(inverse)運算
            反向運算:將0變換成1，1變換成0
    step2-2:再用2進位數字換算成10進位數字
   step3:完成後正負號+大小數字  
```
```
2's complement
   step1:判斷正負
         判斷最左側的第一個數字是1or0，1為負數，0為正數
   step2:計算大小(magnitude)
      step2-1:右側最後1個數字-1
      step2-2:執行反向(inverse)運算
              反向運算:將0變換成1，1變換成0
      step2-3:再用2進位數字換算成10進位數字
   step3:完成後正負號+大小數字
```
